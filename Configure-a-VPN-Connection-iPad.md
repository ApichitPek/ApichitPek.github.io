# 5.5.5 Configure a VPN Connection iPad
+ คุณทำงานเป็นผู้ดูแลระบบรักษาความปลอดภัยด้านไอทีให้กับเครือข่ายองค์กรขนาดเล็กแห่งหนึ่ง คุณเพิ่งตั้งค่าฟีเจอร์ VPN สำหรับการเข้าถึงระยะไกลบนอุปกรณ์รักษาความปลอดภัยเครือข่ายของคุณเพื่อให้คุณและผู้ดูแลระบบคนอื่นๆ สามารถเข้าถึงเครือข่ายได้อย่างปลอดภัย ขณะนี้คุณอยู่ที่บ้านและต้องการเชื่อมต่อ iPad ของคุณเข้ากับ VPN โดย iPad ของคุณเชื่อมต่ออยู่กับเครือข่ายไร้สายที่บ้านของคุณ
+ ในห้องปฏิบัติการนี้ ภารกิจของคุณคือ:
    + เพิ่มการเชื่อมต่อ VPN IPSec โดยใช้ค่าต่อไปนี้
